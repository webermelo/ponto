# Ponto

Minimal Next.js MVP scaffold generated manually.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view the app.

## Scripts

- `npm run dev` – Launches the Next.js development server.
- `npm run build` – Creates an optimized production build.
- `npm run start` – Runs the compiled app in production mode.
- `npm run lint` – Lints the project using Next.js ESLint configuration.
